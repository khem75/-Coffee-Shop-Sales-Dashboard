# ☕ Coffee Shop Sales Dashboard

An interactive Excel dashboard designed to analyze coffee shop sales, customer footfall, product performance, and ordering trends. The dashboard provides actionable insights through dynamic charts, KPIs, and slicers for better business decision-making.

## 📊 Key Features
- Total Sales, Footfall & Average Bill KPIs
- Quantity Ordered by Hour Analysis
- Sales Distribution by Product Category
- Order Size Distribution Analysis
- Store-wise Sales & Footfall Comparison
- Top 5 Products by Sales
- Weekday Order Trend Analysis
- Interactive Month & Day Filters

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
